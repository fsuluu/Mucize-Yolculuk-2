# 🤰 Mucize Yolculuk (Miracle Journey)

**Mucize Yolculuk** is a comprehensive mobile application designed for expectant mothers to track their pregnancy journey, monitor their health, and stay motivated through interactive gamification.

The application aims to guide mothers-to-date with weekly tracking, informative content, and smart health monitoring tools.

---

## 👩‍💻 Project Purpose & Features

This project is an interactive health tracking and education platform developed with **React Native** and **TypeScript**. It focuses on providing a supportive environment for pregnant women through:

### 🌟 Gamification Features
- **Health Buddy:** A live companion that changes its mood based on your daily activity.
- **Daily Spin:** A daily reward system where users can win points to level up.
- **Badge Collection:** Earn unique badges for completing health milestones (e.g., "First Step", "Star Mother").
- **Level System:** Progress through levels to unlock new benefits and titles.

### 🏥 Health Tracking
- **Weekly Pregnancy Progress:** Follow your baby's development week by week.
- **Blood Sugar Monitoring:** Log and track glucose levels with visual charts.
- **Nutrition & Activity:** Assess daily meals and physical activities.
- **Daily Goals:** Stay on track with personalized daily tasks.

## 🚀 Getting Started (Yerelde Nasıl Çalıştırılır?)

### 1️⃣ Prerequisites (Gereksinimler)
Ensure you have the following installed:
- **Node.js** (LTS recommended)
- **npm** or **yarn**
- **Android Studio**
- **JDK 17**
- **Git**

### 2️⃣ Clone the Project (Projeyi Klonlama)
```bash
git clone https://github.com/fatmasulu12/mucizeyolculuk-gebeler.git
cd mucizeyolculuk-gebeler
```

### 3️⃣ Install Dependencies (Bağımlılıkları Yükleme)
```bash
npm install
```

### 4️⃣ Run on Android (Android Platformunu Çalıştırma)
```bash
npx cap sync android
npx cap open android
```

**After Android Studio opens:**
1. Start an **Emulator** or connect a physical device.
2. Click the **Run** button.

**Alternatively, build from terminal:**
```bash
cd android
./gradlew assembleDebug
```
